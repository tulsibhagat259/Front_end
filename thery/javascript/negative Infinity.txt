3. What is negative Infinity? 
ans: Negative Infinity (-Infinity) is a special numeric value that represents a number smaller than any other number.

It is a value less than all other numbers.

It is part of JavaScript’s number system (IEEE 754 standard).

It behaves like a mathematical negative infinity.
