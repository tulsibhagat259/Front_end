3. What are void elements in HTML?

ans:-Void elements are HTML elements that do not have any content — so they don’t need a closing tag.

They are also called empty elements.

The browser understands that these tags stand alone. 


<br>    <!-- break line  -->    <!-- self tag -->
<img src="image.jpg" alt="Picture">   <!-- image -->
<hr>    <!-- horizontal line -->
<input type="text">  <!-- input field -->
<meta charset="UTF-8"> <!-- metadata -->
<link rel="stylesheet" href="style.css"> <!-- link to CSS -->
